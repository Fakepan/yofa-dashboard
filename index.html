☁️ GitHub 雲端同步設定

設定後，資料自動存到你的 GitHub 倉庫，手機、電腦共用同一份資料，任何裝置修改都會自動同步。

第一步：產生 GitHub 存取金鑰（Token）
用瀏覽器開啟：
github.com/settings/tokens/new
Note 填 yofa-dashboard，Expiration 選 No expiration
勾選 repo（第一個大項目打勾即可）
拉到最底按 Generate token，複製產生的 Token
Token 只存在你的裝置（localStorage），不會傳送到任何第三方
✅ 已連接！資料會在每次新增／編輯後自動同步到 GitHub。
📁 本地資料夾備份（第二層保障）

每次修改後自動將 projects.json 存到你電腦指定的資料夾。
需使用 Chrome 或 Edge 瀏覽器，手機不支援。
