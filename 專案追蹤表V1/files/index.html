讓家裡和公司電腦資料自動同步：

  1. 兩台電腦都到 google.com/drive/download 下載 Google Drive 桌面版
  2. 登入同一個 Google 帳號
  3. 在 Drive 中建立資料夾，例如 佑發空調系統
  4. 把這個 HTML 檔放進該資料夾
  5. 雙擊 HTML 即可使用，操作完按「💾 匯出JSON」儲存到同一個資料夾
  6. 到另一台電腦打開同一個 HTML 後，按「📂 匯入資料」載入即可

備註：本程式會把資料自動暫存在本機瀏覽器內，但建議定期匯出 JSON 到 Drive 做為跨電腦備份。
