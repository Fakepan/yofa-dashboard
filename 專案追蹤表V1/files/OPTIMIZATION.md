# 🎯 優化建議 - 按照優先度排列

## 🔴 高優先度（立即可做）

### 1. 結案核對清單 ⭐⭐⭐ 
**重要度**：最高  
**難度**：中等  
**預計時間**：30 分鐘

**功能**：每個案件展開 8 項結案檢查項目
- 勾選項目自動計算進度
- 全勾時自動升級狀態為「待核准」
- 清單項目可客製化

**實現方式**：
```javascript
// 在每個案件中加入 checks 陣列
{
  id: '1150309PSKS6',
  ...
  checks: [false, false, false, ...] // 8 項
}

// Modal 中新增核對清單展開區
// 點擊「結案進度」按鈕展開
```

---

### 2. 搜尋 & 快速篩選 ⭐⭐
**重要度**：高  
**難度**：簡單  
**預計時間**：20 分鐘

**功能**：表格上方加搜尋欄
- 即時搜尋案號、客戶、工程名稱
- 支援多條件篩選（客戶 + 狀態）
- 顯示匹配結果數

**例子**：
> 搜尋「臺灣大學」→ 立即顯示相關案件

---

### 3. 資料驗證 ⭐⭐
**重要度**：中  
**難度**：簡單  
**預計時間**：15 分鐘

**功能**：防止輸入錯誤
- 成本必須 ≥ 0
- 議價 ≤ 報價
- 案號不重複
- 日期不能是未來

---

### 4. Excel / CSV 匯出 ⭐⭐
**重要度**：中  
**難度**：中等  
**預計時間**：45 分鐘
