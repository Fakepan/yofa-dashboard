# 佑發空調-專案結案儀表板 開發路線圖

## 🎯 專案概述
- **主要檔案**：`index.html`（獨立網頁應用）
- **技術棧**：純前端 HTML/CSS/JavaScript
- **儲存方式**：localStorage（瀏覽器本機儲存）
- **目標**：完整的專案追蹤與淨利分析系統

---

## 📋 已實現的功能（v1.0）

✅ 8 筆初始案件資料  
✅ 4 大分析圖表（每月案件、營收、淨利、獲利排名）  
✅ 自動扣稅 5%，計算淨利和毛利率  
✅ 成本直接編輯  
✅ PDF 上傳/下載  
✅ 案件新增/編輯/刪除  
✅ 資料匯出/匯入 JSON  
✅ 本機自動儲存（localStorage）  
✅ 響應式設計（桌面 + 手機）

## 💡 具體改進建議

### 1️⃣ 新增結案核對清單
```javascript
// 8 項結案檢查項目
const CHECKLIST = [
  '審查專案計劃與預算',
  '更新施工完工文件',
  '取得客戶完工確認簽章',
  '索取客戶核准/驗收單',
  '確認款項已收訖',
  '歸檔估價單與相關圖面',
  '舉行專案後內部檢討',
  '交接剩餘材料與庫存'
];
```
- 每案 1 個可展開的核對清單
- 勾選項目 → 進度條更新
- 全勾 → 自動升級「待核准」

### 2️⃣ 搜尋欄位（表格上方）
- 即時搜尋案號 / 客戶 / 工程名稱
- 顯示匹配結果數量
- 按 Enter 或按鈕搜尋

### 3️⃣ Excel 匯出
```javascript
// 需要引入 SheetJS 庫
// 一鍵產生 .xlsx 檔案，包含：
// - 案件列表（含淨利、毛利率）
// - 月份統計表
// - 客戶統計表
```

### 4️⃣ 客戶面板（新 Tab）
- 獨立統計各客戶的總金額、案件數
- 平均毛利率排排名
- 客戶聯絡窗口備註

### 5️⃣ 數據驗證 & 錯誤提示
- 成本不能是負數
- 議價金額不能超過報價金額
- 日期不能是未來
- 案號重複警告
